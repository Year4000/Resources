--- title: Getting Started layout: default weight: 0 --- ### Creating a Map You can create a map like you would do anyways, for our system MapNodes, to work with the map, it must read a file named `map.json` inside of the world folder. You may also include a 512x512 png names `icon.png` and this will be used for the ping, as used on our website. We use [JSON](http://www.json.org/){:target="_blank"} to parse the maps. JSON is a simply to learn data interchangeable format. This allows us to do more than `YML` and `XML`, but still easy to learn and read. Take a look at it for help. The `map.json` has several components that make MapNodes work. Some of the parts are required and some are not. This section will tell you how each component works and if they are required or not. #### Folder Structor When you are done making the map it should follow this structure. You can see how the current maps work [here](https://maps.year4000.net/){:target="_blank"}.
