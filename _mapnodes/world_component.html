This component will control various aspects of the world for each map. They do not rely on a specific game mode and can be used for any map.

Field Type Description Default
spawn Region Type The url to download a resource pack to the clients. Required
difficulty Integer /
String		 The world's difficulty level, allows for string presets peaceful, easy, normal, and hard. Hard
weather Boolean The world's forced weather, sun false or rain true. False
time_lock Integer /
String		 The time the world will be forced to, allows for string presets noon, midnight, dusk, and dawn<./td> None
environment Environment The world's enviroment normal, nether, the_end. Normal
world_height Integer The max height that you can place blocks. 256
resource_pack URL The url to download a resource pack to the clients. None
