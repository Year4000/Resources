Games this controls what game modes are enabled and what the settings are for them. To enable a game mode you just add another object the this map. Each key is the game mode's name and the value object are the settings for that game mode. You can have more than one game mode but this may cause unexpected results.

{% highlight json linenos %} { "games": { } } {% endhighlight %}

Game Modes

Bellow are the various game modes that you can apply to any map.

  • Capture
  • Deathmatch
  • Skywars
  • TNT Wars

Capture

Capture is a gamemode where you have to grab a block from one side of the map and bring it back to your base. Their are predefined capture variables that you can set and it will pre fill out the Name, Block, and Data.

Field Type Description Default
block_captures List The block capture objects that describe the capture block. Required
{% highlight json linenos %} { "capture": { "block_captures": [ ] } } {% endhighlight %}
Field Type Description Default
owner String The team id that owns the current block. Required
challenger String The team id that needs to capture the block. Required
region String The region id that the block point will be at. Required
name String The name that will show up on the scoreboard. Required
block String The item type of the block that must be captured. Required
data Integer The data of this custom block. Required
wool ChatColor The chat color of the wool block. Required
clay ChatColor The chat color of the wool block. Required
NOTE: If the wool variable is set you can not set name, block, and data as they will be overwriten.
NOTE: If the clay variable is set you can not set name, block, and data as they will be overwriten.

Deathmatch

Deathmatch is a gamemode where you are given points for killing players on the other teams. To win you must run time out or reach the max score. With the addition that you can add point boxes to help aid the teams score.

Field Type Description Default
time_limit Duration Play untill the time runs out can be used with max_score. None
max_score Integer The max score the team needs to reach to end the game. None
kill_point Integer The points given to the challenger's team. 1
point_boxes List A list of point box objects. None
{% highlight json linenos %} { "deathmatch": { "time_limit": "10m", "max_score": "100", "kill_point": "2", "point_boxes": [ ] } } {% endhighlight %}
Field Type Description Default
owner String The team id that owns the point box. Required
challenger String The team id that needs to enter the point box. Required
region String The region id that the point box will be at. Required
point Integer The points given to the challenger's team. 1
{% highlight json linenos %} { "owner": "", "challenger": "", "region": "", "point": 5 } {% endhighlight %}
Skywars

Skywars is a gamemode where you fight to the death, with islands in the sky. Though due note that the player's teams' spawns, needs to match the team's max size and have one point for each island.

Field Type Description Default
players_team Duration The team id that will act as the list of players. Required
{% highlight json linenos %} { "skywars": { "players_team": "" } } {% endhighlight %}
TNT Wars

TNT Wars is a game where you are to make TNT Cannons and blast them to the other team isalnd to destroy it and try to kill players while you do it. But their is a wall guarding the islands you must find a way to get around it.

Field Type Description Default
time_limit Duration The max time players have to destroy islands to win the game. Required
max_score Integer The score needed to win the game, points are given by kills and island destuction. Required
islands List The list of island objects to define what island belongs to whom. Required
{% highlight json linenos %} { "tnt_wars": { "time_limit": "1h", "max_score": 1500, "islands": [ {"owner": "", "region": ""}, {"owner": "", "region": ""} ] } } {% endhighlight %}
Field Type Description Default
owner String The team id that this island belongs to. Required
region String The region id this island is for. Required
{% highlight json linenos %} { "owner": "", "region": "" } {% endhighlight %}
