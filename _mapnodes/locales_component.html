The locales are not required if you use the built in locales. Though if you do not use the builtin locales, it is easy to add locales for your map.

Where you can place locales

  • In the map's description field.
  • Item's name field.
  • Item's lore field.
  • Books's title field.
  • Books's page fields.
  • Region's message field.
