This component is required, this tells MapNodes information about the map. It includes a name, version, description, authors.

Field Type Description Default
name String The map name, it should be less than 22 chars Required
version String The map's version, it should in the standard format of MAJOR.MINOR (1.0) Required
description String What are the goals of the map. Required
authors List A list of the authors uuids that helped make the map. Use mcuuid.ewized.com to get the uudis of players. Required
