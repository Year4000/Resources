Teams are not only teams but groups. This controls how teams work. The teams component is a Map of teams, each team will have a key, this key is used to id the team for various gamemodes and internal use.

Field Type Description Default
name String The team's display name, it differs from the team's key id. Required
color ChatColor The color of the team, used in the scoreboards. white
kit String The id of the kit the team will get when they spawn in. default
size Integer The max size of the team. 1
spawns List The list of region types. Required
friendly_fire Boolean Allow team members to attack each other. False
friendly_invisibles Boolean Allow team members to see transparent players when they are invisibile. True
